# Web前端刷题助手 — 文件架构

## 目录结构

```
project/
├── index.html                   入口 HTML，Vite 挂载点
├── package.json                 项目依赖（vue、vue-router、typescript、vite）
├── vite.config.ts               Vite 配置（@ 别名、自动打开浏览器）
├── tsconfig.json                TypeScript 配置入口
├── tsconfig.app.json            TS 应用配置（@/* 路径映射）
├── tsconfig.node.json           TS Node 配置
│
├── public/
│   └── favicon.ico              网站图标
│
└── src/
    ├── main.ts                  应用入口，创建 app、注册 router、挂载到 #app
    ├── App.vue                  根组件，<router-view /> + 全局样式重置
    │
    ├── types/
    │   └── index.ts             TS 接口：Question（题目）、UserAnswer（作答记录）
    │
    ├── data/
    │   └── questions.ts         12 道模拟题数据（4 知识点 × 3 题）
    │
    ├── router/
    │   └── index.ts             路由配置：/ /quiz /result，含 beforeEnter 守卫
    │
    ├── views/                   页面级组件
    │   ├── WelcomeView.vue      欢迎页：标题 + "开始刷题"按钮
    │   ├── QuizView.vue         刷题页：组装 QuizProgress + QuestionCard，管理答题状态
    │   └── ResultView.vue       结果页：组装 ResultSummary + 逐题展示对错和解析
    │
    └── components/              可复用组件
        ├── QuestionCard.vue     单题卡片：题干 + ABCD 选项（props/emit）
        ├── QuizProgress.vue     顶部分类标签 + 答题进度（已答 N/12）
        └── ResultSummary.vue    成绩统计面板：答对数/答错数/正确率
```

## 组件职责表

| 组件 | 类型 | Props（接收） | Emit（发出） | 核心职责 |
|------|------|--------------|-------------|---------|
| **WelcomeView** | 页面 | — | — | 展示项目标题，点击按钮跳转 `/quiz` |
| **QuizView** | 页面 | — | — | 持有全局答题状态 `userAnswers`，组装子组件，处理提交 |
| **ResultView** | 页面 | — | — | 从 router state 取答案数据，computed 统计对错，逐题展示解析 |
| **QuestionCard** | 组件 | `question`、`selectedIndex` | `update:selectedIndex` | 渲染单题题干和 4 个选项，用户点击通知父组件 |
| **QuizProgress** | 组件 | `categories`、`currentCategory`、`answeredCount`、`total` | `update:currentCategory` | 渲染分类标签，点击切换筛选；显示答题进度 |
| **ResultSummary** | 组件 | `correctCount`、`wrongCount`、`accuracy`、`total` | — | 纯展示统计数字（答对/答错/正确率） |

## 数据流

```
questions.ts (12条静态数据)
       │
       ▼
  QuizView.vue (唯一状态持有者: userAnswers[])
       │
       ├── props ──→ QuizProgress.vue (分类、进度)
       │   ← emit ── 点击标签 → 切换分类筛选
       │
       └── props ──→ QuestionCard.vue ×12 (题目、选中值)
           ← emit ── 点击选项 → 更新 userAnswers
       │
       ▼ router.push('/result', { state })
       │
  ResultView.vue
       │
       ├── props ──→ ResultSummary.vue (统计数字)
       │
       └── v-for ──→ QuestionCard.vue ×12 (展示模式 + 解析)
```

## 路由表

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | WelcomeView | 项目首页，开始刷题入口 |
| `/quiz` | QuizView | 刷题训练页，12 道选择题 |
| `/result` | ResultView | 成绩结算页，含 beforeEnter 守卫 |

## 必做项覆盖情况

| # | 要求 | 实现位置 |
|---|------|---------|
| 1 | Vue3 + TypeScript | 全项目 |
| 2 | Vue Router | `router/index.ts` |
| 3 | ≥3 页面 | Welcome / Quiz / Result |
| 4 | ≥1 列表页 | QuizView（题目列表）、ResultView（结果列表） |
| 5 | ≥8 条数据 | `questions.ts`（12 条） |
| 6 | v-for | QuestionCard 选项、QuizProgress 标签、QuizView 题目、ResultView 结果 |
| 7 | v-model | QuestionCard 选项绑定 |
| 8 | computed | ResultView 统计、QuizView 筛选 |
| 9 | props / emit | QuestionCard、QuizProgress |
| 10 | ≥3 组件 | QuestionCard / QuizProgress / ResultSummary |
| 11 | 路由守卫（加分） | `/result` beforeEnter |
